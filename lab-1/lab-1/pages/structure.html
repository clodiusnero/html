How to use HTML5 structural elements

Section
Tag contains different group of artickles.

Article
Tag contains an article, stand-alone piece of information.

Header
Tag contains the head-content of a site. Namley top level with logo.

Footer
Tag contains the foot-content of a site.

Nav
Tag contains the main menu for the page.

Aside
Contains a section that is not a part of the main content, but it's related, usually placed left or right of the main content.
